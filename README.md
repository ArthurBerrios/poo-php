# Projeto de Revisão - Programação Orientada a Objetos em PHP

Este projeto tem como objetivo revisar e aplicar os principais conceitos de **Programação Orientada a Objetos (POO)** em PHP, com uma estrutura de pastas organizada, utilizando **autoloader** e boas práticas de codificação.

## Conceitos Abordados

- ✅ **Namespaces**  
  Organização das classes em namespaces, melhorando a estrutura e evitando conflitos de nomes.

- ✅ **Traits**  
  Compartilhamento de funcionalidades entre classes com reuso de código (ex: cálculo de datas).

- ✅ **Interfaces**  
  Definição de contratos que garantem métodos obrigatórios nas classes que as implementam.

- ✅ **Herança**  
  Reaproveitamento de código através da extensão de classes.

- ✅ **Polimorfismo**  
  Comportamento diferente para o mesmo método, dependendo da classe que o implementa.

- ✅ **Autoload **  
  Utilização do autoloader para carregamento automático de classes.

- ✅ **Classes Abstratas**  
  Criação de classes base que não podem ser instanciadas diretamente, mas servem de modelo para outras.
